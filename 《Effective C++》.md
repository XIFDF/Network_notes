# 《Effective C++》读书笔记
## 摘自《Effective C++》第三版
## 条款01：视C++为一个语言联邦
### View C++ as a federation of language.
关注C++的四个次语言：
* C：C语言
* Object-Oriented C++：面向对象
* Template C++：泛型编程
* STL
## 条款02：尽量以 const，enum，inline 替换 #define
### Prefer const,enums,and inlines to #defines.
对于单纯常量，最好以const对象或enums替换#define。
